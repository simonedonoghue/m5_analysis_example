M5 Exposure Analysis
--------------------------------

The script in this folder computes treatment exposure to the Milan M5 metro line for census tracts (year 2011)

Structure:
- data/: input shapefiles
- output/: exported spatial datasets
- m5_script_example.R: main script

Instructions:
1. Open the project in RStudio.
2. Install required packages if needed.
3. Run m5_script_example.R.

Required packages:
sf, tidyverse, tmap, units, here
